Key Features

A full-cycle TA recruitment workflow with intelligent matching

🔬

Explainable Recommendation

6-dimensional weighted scoring engine: skill match, availability, experience, workload balance, profile evidence, and competition pressure. Every score comes with human-readable explanations.

👥

Role-Based Dashboards

Three distinct dashboards for TA, Module Organiser (MO), and Admin. Each role sees relevant data, actions, and KPIs tailored to their workflow.

📄

Complete Application Lifecycle

Apply → Shortlist → Accept/Reject → Withdraw. Full state machine with automatic job closure when quota is met or deadline passes.

💼

Job Management

MOs can create, edit, close, and reopen positions. Input validation prevents XSS, enforces deadlines, and ensures quota consistency.

📊

Workload Monitoring

Admin dashboard tracks TA workload balance across all accepted positions. Alerts when threshold is exceeded for fair distribution.

💾

Zero-Database Architecture

All data persisted as JSON/CSV/TXT files. Built-in caching with path-level read-write locks for concurrent access. No database setup required.

Recommendation Engine

Multi-dimensional explainable scoring with configurable weights

The engine evaluates each TA-Job pair across 6 dimensions. Weights are configurable in data/system/config.json.

Formula: score = Σ(dimension_score × weight) / Σ(weight)

40%

🎯 Skill Match

Jaccard similarity + alias normalization (e.g., "OOP" = "Object Oriented Programming"). Required coverage ×0.72 + preferred ×0.18 + breadth bonus + full coverage bonus − gap penalty.

18%

📚 Experience Evidence

Analyses profile text, CV content, and experience descriptions. Combines phrase coverage, token overlap, and evidence keyword matching (lab, marking, debugging, etc.).

12%

🕐 Availability

Keyword detection in availability text: weekday/weekend, specific days, time slots (morning/afternoon/evening), flexibility indicators.

12%

⚖ Workload Balance

Evaluates current TA workload against configurable threshold (default 12h/week). Over-threshold candidates get penalty scores to promote fair distribution.

10%

📋 Profile Evidence

Assesses profile completeness (5 fields) weighted at 58%, job alignment at 22%, and skill breadth signal. Encourages TAs to fill comprehensive profiles.

8%

🏅 Competition Pressure

Calculates active-applicants-per-remaining-slot ratio. Low competition (≤1/slot) scores 0.95; high competition (>4/slot) scores progressively lower.
